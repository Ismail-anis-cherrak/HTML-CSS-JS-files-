Table Element Examples

The <table> element in HTML is used for creating tables. It can include elements like <thead> (table header), <tbody> (table body), <tfoot> (table footer), <tr> (table row), <th> (table header cell), and <td> (table data cell).

The border attribute is used to specify the border width of the table and its cells, but it's not recommended in modern web design. Instead, CSS should be used for styling.
