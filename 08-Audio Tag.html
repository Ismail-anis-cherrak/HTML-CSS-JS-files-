HTML Audio Tag Examples

The <audio> tag in HTML is used to embed audio content on a web page. It supports various attributes to control the playback and appearance of the audio player. Here are examples for each attribute:

Example 1: Basic Audio with Controls

The controls attribute adds playback controls to the audio player, allowing users to play, pause, adjust the volume, and more.

Example 2: Autoplay

The autoplay attribute automatically starts playing the audio when the page loads. Please note that this can be intrusive, so use it carefully.

Example 3: Loop

The loop attribute causes the audio to continuously loop, restarting when it reaches the end.

Example 4: Muted

The muted attribute starts the audio playback with the sound muted.

Example 5: Multiple Sources with Different Formats

Provide multiple <source> elements to ensure compatibility with different browsers and audio formats.
